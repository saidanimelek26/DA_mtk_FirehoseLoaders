📖 About This Project

This repository contains reverse engineering analysis of MediaTek preloader and LK (Little Kernel) for FRP (Factory Reset Protection) mechanism.


KMAC128 AES-ECB RSA-4096 EFUSE OEMINFO MediaTek MT6768

⚡ Quick Commands

# Clone the repository
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git

# Run FRP analysis
python3 preloader_analyzer.py preloader_k69v1_64_64only.bin

# Extract crypto keys
python3 frp_crypto_extractor.py lk.img
